### Authentication Notes

**slack-term:**
- Issue #284: 'Token Method 1 no longer works'
- Manual browser token extraction required for newer workspaces
- Extract xoxc- from LocalStorage and xoxd- from Cookie

**slackcat:**
- OAuth configuration via --configure
- Supports both legacy and OAuth tokens

**slack-cli:**
- Relies on legacy tokens (deprecated by Slack in 2020)
- Likely incompatible with current Slack API

**slack-libpurple:**
- Supports multiple auth methods including browser token
- 2FA incompatible (Issue #115)
- May require manual token extraction for some workspaces

**slack-cli:**
- Maintenance mode limits auth updates
- May have compatibility issues with evolving Slack auth

**yfiton:**
- Each service has its own auth configuration
- Slack integration via -n slack parameter

**slack-cli:**
- Standard OAuth flow for app development
- Service token support for CI/CD

**slack-cli:**
- Relies on outdated Slack API patterns

**slackcli:**
- Dual auth: standard OAuth + browser tokens
- Browser auth bypasses complex OAuth flows
